# Python DevContainer

Python development environment with uv package manager optimized for Claude Code.

## Features

- Python development environment with uv package manager
- Fast, reliable Python package management with uv
- Pre-installed development tools:
  - `uv` - Fast Python package installer and resolver
  - `uvx` - Run Python applications in isolated environments
- Zsh shell with Powerline10k theme
- Claude Code integration

## Usage

Copy the `.devcontainer` folder to your Python project root:

```bash
cp -r .devcontainer /path/to/your/python-project/
```

Then open your project in VS Code and select "Reopen in Container".
